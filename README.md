
Create a folder anywhere, and paste the following to db.json <br/>
`{
  "tasks": [],
  "comments": [],
  "labels": [],
  "projects": [],
  "users": []
}` <br/>
Then start json-server `json-server --watch db.json --port 3001`<br/>

In the project directory, you can run:

# `npm install`
# `npm start`
